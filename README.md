# EffOff  
### Automatic Efficiency Mode Watchdog for Windows 11

## 🚨 Is Your PC Slow Even When CPU Usage Is Low?

Windows 11 may automatically enable **Efficiency Mode**, lowering process scheduling priority — even when your system isn't under heavy load.

This can cause:

- Sluggish UI
- Input lag
- Random performance drops
- Background apps responding slowly
- Games stuttering despite low CPU usage

Many users manually disable Efficiency Mode in Task Manager —  
but Windows may re-enable it automatically.

---

## 💡 What EffOff Does

EffOff is a lightweight background utility that:

✔ Monitors all processes in real time  
✔ Automatically disables unwanted Efficiency Mode  
✔ Starts with Windows  
✔ Requires no manual intervention  
✔ Uses minimal system resources  

It acts as a watchdog for Windows scheduling behavior.

---

## 🔬 Why This Matters

Efficiency Mode reduces CPU scheduling priority.

Even when CPU usage is low (e.g., 5%),  
Windows may still throttle certain processes.

This can create the illusion of "low usage but slow performance."

EffOff restores full scheduling priority automatically.

---

## 🔒 Privacy & Security

- No telemetry  
- No data collection  
- No network communication  
- No background tracking  

EffOff runs locally on your system only.

# EffOff  
### Windows 11 효율성 모드 자동 감시 및 해제 도구

## 🚨 CPU 사용률이 낮은데도 PC가 느린가요?

Windows 11은 자동으로 **효율성 모드(Efficiency Mode)** 를 활성화할 수 있습니다.

이 기능은 프로세스의 스케줄링 우선순위를 낮춰  
CPU 자원 사용을 줄이는 대신, 반응 속도를 저하시킬 수 있습니다.

그 결과:

- UI가 버벅거림
- 입력 반응 지연
- 원인 모를 성능 저하
- 백그라운드 앱이 느려짐
- CPU 사용률은 낮은데 체감 속도는 느림

작업 관리자에서 수동으로 끌 수는 있지만,  
Windows가 다시 자동으로 활성화하는 경우가 있습니다.

---

## 💡 EffOff는 무엇을 하나요?

EffOff는 Windows 11의 효율성 모드를 실시간으로 감시하고  
원하지 않는 자동 활성화를 즉시 해제하는 경량 유틸리티입니다.

✔ 모든 프로세스를 실시간 모니터링  
✔ 자동으로 효율성 모드 해제  
✔ Windows 시작 시 자동 실행  
✔ 사용자 개입 없이 동작  
✔ 매우 낮은 시스템 자원 사용  

EffOff는 Windows 스케줄링 동작을 감시하는 **Watchdog 도구**입니다.

---

## 🔬 왜 이게 중요한가요?

효율성 모드는 CPU 사용률과 별개로  
프로세스의 스케줄링 우선순위를 낮출 수 있습니다.

즉,

CPU 사용률이 5%여도  
Windows가 특정 프로세스를 낮은 우선순위로 처리하면  
실제 체감 성능은 저하될 수 있습니다.

EffOff는 이러한 자동 스로틀링을 감지하고  
정상 우선순위를 복원합니다.
